Technologies

Among internet font-end technologies we work with there are standards like (X)HTML, CSS (Cascading Style Sheets), JavaScript and its frameworks (mostly jQuery, jQuery mobile, Prototype and Modernizr), Flash / Actionscript 3 and new emerging standards like HTML5 and CSS3.

Find out more:

Flash and Actionscript 3 technologies allow to build encapsulated web content, enriched with animations, transitions and video. Flash used with Actionscript 3 object-oriented programming is the best way to present rich media online. Flash is also leading technology in website banner ads. For now, Flash technology is poorly supported by handheld devices.

JavaScript is a programming language that in web design describes behaviors and dynamic ux elements, often used to boost up user interaction and send server queries. JavaScript frameworks, such as jQuery, prototype or Modernizer serve as great tools to build animations, transitions and as a way of abstraction layer built to eliminate differences between browsers' rendering.

HTML5 and CSS3 are emerging technologies, not fully implemented in all browsers yet, but already in wide use because of their capabilities - new types of mark-up elements with new functionality like geolocation, local storage, video and sound embeding and styles targeted for mobile devices.
